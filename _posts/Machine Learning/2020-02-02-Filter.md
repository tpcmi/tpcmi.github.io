## 高斯滤波·Gassian Filter

线性平滑滤波，适用于消除高斯噪声[^1]，就是对整幅图像进行加权平均的过程，每一个像素点的值，都由其本身和邻域内的其他像素值经过加权平均后得到，具体操作是：用一个模板（或称卷积、掩模）扫描图像中的每一个像素，用模板确定的邻域内像素的加权平均灰度值去替代模板中心像素点的值。

**滤波是掩膜（mask）和图像的卷积**

- 计算掩膜

利用高斯分布公式计算掩膜，原先的高斯公式是一维的，但是将x和μ换成向量，即可以表示二维的坐标值，此处μ为掩膜中心坐标（**统计学中称为均值，即坐标的均值**），x为掩膜内任一点的坐标

| （x-1,y-1） | （x,y-1） | （x+1,y-1） |
| :---------: | :-------: | :---------: |
|  （x-1,y）  |  （x,y）  |  （x+1,y）  |
| （x-1,y+1） | （x,y+1） | （x+1,y+1） |

$(\overrightarrow{x}-\overrightarrow{μ})^2$

|  2   |  1   |  2   |
| :--: | :--: | :--: |
|  1   |  0   |  1   |
|  2   |  1   |  2   |

其中x，y在遍历的过程中抵消了。

∵ $e^{-\frac{(\overrightarrow{x}-\overrightarrow{μ})^2}{2σ^2}}$当σ=1时:

| 0.3679 | 0.6065 | 0.3679 |
| :-: | :-: | :-: |
| 0.6065 | 1.0000 | 0.6065 |
| 0.3679 | 0.6065 | 0.3679 |

再进行一次归一化，使核中所有值相加为1即可用于卷积。

- 卷积

以3✖️3的核为例，正常每一次卷积需要做9次乘法，8次加法，而可以采取如下加速改进    1）将2维高斯滤波分离卷积为3✖️1和1✖️3的一维高斯滤波，那么每次需进行6次乘法和5次加法；

2）掩膜的系数是固定的，以上述核为例，只有三个不同参数，而像素值的范围也是有限的，只有0~255，256个值，那么系数和像素值的乘积只有768个值，后续在进行运算时，可以直接查表，将整体运算简化为5次加法运算。

## 各向异性扩散滤波·Anistropic Filter

克服了高斯模糊的缺陷，具有保留图像边缘细节同时减少噪声的作用









[^1]: 指噪声的概率密度函数服从高斯分布，即正态分布的一类噪声，抑制方法常常采用数理统计方法。通常是由于不良照明或高温引起的传感器噪声，在数字图像处理中，可以使用空间滤波器来降低高斯噪声，但是当对图像进行平滑时，结果可能导致精细缩放的图像边缘和细节的模糊，因为它们也对应于被阻挡的高频。 用于噪声去除的常规空间滤波技术包括：平均（卷积）滤波，中值滤波和高斯平滑。
